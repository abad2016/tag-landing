Una comunidad en la que gamers, developers y event managers pueden tener una relación mas unida.

Acerca

El grupo IGNIS presenta su último proyecto TAG. Together As Gamers. Una aplicación pensada para unir a la comunidad Gamer. Buscamos entre todo una mejor oportunidad para todos, tanto gamers, para encontrar nuevos juegos y para developers y event managers, para sacar sus ideas al mundo.

Beneficios

Chat instantáneo entre Gamers, Developers y Event Managers. Haciendo la comunicación entre todos mas rápida y fácil.

Facilidad para los Gamers de encontrar nuevos juegos ajustados a sus gustos. Podrán ayudar a los game developers a desarrollar sus juegos, probando sus ultimas versiones.

Game Developers podrán desarrollar sus juegos y obtener mucho mas feedback, construyendo mejores juegos gracias a la comunidad.

Nuevos eventos todos los días y al alcance de todos. Gran facilidad de realizar tus propios eventos!
